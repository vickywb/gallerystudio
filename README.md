# Gallery Studio

Ini adalah aplikasi web pembelajaran menggunakan **Laravel 11** untuk portfolio dan e-commerce studio kreatif. Web ini menggabungkan sistem manajemen konten (CMS) untuk portfolio dan blog dengan fitur e-commerce untuk menjual paket layanan.

## 📌 Teknologi Utama

- **Framework**: Laravel 11
- **Database**: MySQL/SQLite
- **Payment Gateway**: Xendit
- **Frontend**: Vite + HTML/CSS/JS
- **Authentication**: Laravel Auth + Spatie Permission (Role-based)
- **Validasi**: CAPTCHA pada form kontak

## 🎯 Kegunaan Web

Web ini digunakan untuk:

- **Showcase Portfolio** - Memamerkan galeri karya dengan kategori
- **Blog Management** - Menulis dan mengelola artikel
- **Menjual Paket** - Menawarkan dan menjual paket layanan dengan sistem pembayaran terintegrasi
- **Customer Management** - Tracking transaksi dan manajemen customer
- **Admin Panel** - Dashboard lengkap untuk mengelola semua konten
